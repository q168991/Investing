Hi Mark!

Value investing is the practice of buying a stock below it's "intrinsic value" in the hopes that it will eventually rise to that intrinsic value.

The "intrinsic value" is what a value investor thinks the company is actually worth. For example, if a stock's price is currently $30, but a value investor, after having looked at it's financial statements, thought the stock is worth $50, then $30 would the the actual value and $50 would be the intrinsic value.

Value investing is a little tricky, in that there isn't any set-in-stone way to determine intrinsic value. It varies from person to person.
